# 🍎 Dashboard de Vendas - Apple (Análise Comercial)

### 📌 Sobre o projeto
Este dashboard foi desenvolvido para monitorizar e analisar o desempenho comercial de uma linha de produtos tecnológicos. O foco principal é fornecer uma visão clara sobre o faturamento, volume de vendas por categoria e a eficiência logística através do controlo de devoluções.

---

### 🚀 Tecnologias Utilizadas
* **Power BI**: Para a criação de visualizações interativas e intuitivas.
* **DAX (Data Analysis Expressions)**: Para o cálculo de métricas de faturamento e taxas de retorno.
* **Mapas Interativos**: Para análise da distribuição geográfica das vendas por loja.

---

### 🎯 Indicadores Principais (KPIs)
O painel destaca métricas fundamentais para a gestão de vendas:

1. **Faturamento Total**: Visualização rápida do montante global acumulado (99,3 Mi).
2. **Performance por Produto**: Identificação do carro-chefe (iPhone com 9,0 mil unidades).
3. **Logística Reversa**: Controlo rigoroso da quantidade devolvida por produto para identificar possíveis problemas de qualidade ou insatisfação.
4. **Análise Mensal**: Gráficos de barras que mostram a sazonalidade e evolução do faturamento ao longo dos meses.
